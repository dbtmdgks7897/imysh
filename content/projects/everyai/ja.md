## 1. 背景 — WHY

### テーマ

AIを活用して制作した様々な作品やサービスを取引できるプラットフォーム。

### 目標

- AI作品を取引できるようにすることで、消費者は合理的な価格で必要なサービスを受けられる
- AI技術を活用し、新旧の技術者が新たな収益化を実現できる

## 2. プロセス — HOW

### 使用技術スタック

**フロントエンド:**
- Next.js
- HTML, CSS, JS

**バックエンド:**
- Java (JDK 17)
- Spring Boot (3.1.4)
- JPA
- Spring Security

### 設計

実務プロジェクトのため、DB・画面設計の詳細は非公開です。

> [スクリーンショット: パスワード変更 (비밀번호_변경.gif)]

- **ログイン（JWT）** — バックエンドでJWTを発行してフロントに返し、Cookieに保存。以後、認証が必要なリクエストの際にCookieからJWTを送信し、JwtFilterでトークンを検証。[(velog リンク)](https://velog.io/@dbtmdgks7897/Spring-Boot-Jwt-3-%EA%B5%AC%ED%98%84-%ED%9A%8C%EC%9B%90%EA%B0%80%EC%9E%85-%EB%A1%9C%EA%B7%B8%EC%9D%B8)

> [スクリーンショット: ログイン JWT (로그인(JWT).gif)]

- **再ログイン** — accessTokenの有効期限1日、refreshTokenは7日に設定。accessToken期限切れ時は401エラーを返し、refreshTokenで再ログインをリクエスト。refreshTokenが期限切れまたは無効な場合はログイン画面へ遷移。

- **ソーシャルログイン** — ソーシャルログイン情報をもとにサーバーでデータの有無を確認し、会員登録またはログイン処理
